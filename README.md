Overview
========

Fishing e-commerce web application for borntofish.org .

It allows you to buy cool fishing stuff under the borntofish.org company.

Technical Stack
~~~~~~~~~~~~~~~

* PostgreSQL
* Python
* Django
* Twitter Bootstrap
* jQuery
